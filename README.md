## Projects

### calc/
A simple command-line calculator built while learning Python basics.

**Features:**
- Basic arithmetic (+, -, *, /)
- Loop for multiple calculations
- Quit option ('q')
- Exception handling (invalid numbers, division by zero)
- Input validation for operators

**Version 3.0 (17th may 2026):**
- Square root calculation ('r)
- Percentage calculation ('%' operator: x% of y)
- Memory sotre ('ms') and recall ('mr')

**Concepts practiced:**
- `while` loops with `break`
- `if`/`elif`/`else` flow control
- `try`/`except` for exception handling (`ValueError`, `ZeroDivisionError`)
- Type conversion (`int()`)
- User input with `input()`
- `logging` module with multiple levels (debug, info, warning, error)
- f-strings for cleaner string formatting
- Logging to file with `basicConfig(filename=...)`
- `.gitignore` to exclude generated log files from version control
